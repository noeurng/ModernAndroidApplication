# ModernApp
Best practices from Android development world<br>
* Language: Kotlin
* Architecture: Clean Architecture
* Android: SupportLibrary, CardView, Recycler, ConstraintLayout, ArchitectureComponents
* Async: Coroutines
* Network: OkHttp, Retrofit
* Database: Room
* DI: Koin
